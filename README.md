## Como executar o projeto
```bash
    # clone o projeto
    git clone https://github.com/matheuspeluso/Quiz.git

    #entrar na pasta do projeto
    cd Quiz

    #instalação das dependencias usando npm 
    npm install 

    #para executar o sistema 
    npm run dev
    
```
